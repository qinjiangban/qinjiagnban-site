查看 npm 的版本
```bash
npm -v
```

查看全局已安装
```bash
npm ls -g
```

所有包的依赖显示
```bash
npm ls --depth 0
```


查看项目中已安装
```bash
npm ls
```

查看详细的漏洞信息
```bash
npm audit
```

自动修复大多数漏洞
```bash
npm audit fix
```




查看依赖更新对比
```bash
npm outdated
```

更新到最新模块
```bash
npm i next@latest
```

更新模块
```bash
npm update $
```


卸载模块
```bash
npm uninstall $
```
