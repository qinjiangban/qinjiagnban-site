
| 命令行按下|  功能                 |
| ------:   | ---                   |
| `a`       | 打开安卓              |
| `shift+A` | 选择设备或仿真器      |
| `w`       | 打开卷筒纸            |
| `r`       | 重新加载应用程序      |
| `j`       | 打开调试器            |
| `m`       | 切换菜单               |
| `shift+M` | 更多工具              |
| `o`       | 在编辑器中打开项目代码 |
| `c`       | 显示项目二维码         |
